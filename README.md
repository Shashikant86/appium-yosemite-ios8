BDD OSX Yosemite Appium iOS8
========================

## Install

Requirements

*  Mac OSX with Yosemite 
* Xcode 6
* Ruby-Cucumber
* NodeJS
* BDDfire [Optional]

Your description to install project

            $ bundle install 
 Make sure you are using corect ruby version. 193            

## Usage

### Install Node 

           $ npm install 

### Run cucumber bundle tasks

             $ bundle exec cucumber -p appium_ios
